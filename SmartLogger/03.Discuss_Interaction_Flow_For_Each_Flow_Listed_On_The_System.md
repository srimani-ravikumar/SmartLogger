### 1. Basic Logging Flow

* **Create:** Application generates log message.
* **Process:** Logger receives message.
* **Filter:** Logger checks if message level is enabled.
* **Route:** Logger sends valid messages to output destinations.
* **Write:** Each destination records the message.

---

### 2. Configuration Flow (Real-time)

1. **Input:** Application updates `LogConfiguration`.
2. **Update:** Logger applies new settings immediately.
3. **Effect:** All subsequent logs use the updated rules.

---

### 3. Multi-threaded Flow

* **Concurrency:** Multiple threads send messages at the same time.
* **Safety:** Thread-safe Logger sequences the requests.
* **Execution:** Destinations handle writes without data corruption.

---

### 4. Formatting Flow

* **Receive:** Message hits the destination.
* **Transform:** Destination applies specific formatting (e.g., JSON, Text).
* **Finalize:** Formatted string is written to the final output.

---

### 5. Logger Initialize Flow
