* Log Levels  
	- DEBUG, INFO, WARNING, ERROR, FATAL: ENUM (for better extensibility)
	- IsGreaterOrEqual(LogLevel level): boolean

* LogMessage Structure
	- timestamp: Timestamp
	- level: LogLevel
	- message: string (Detailed message to be logged)
	- source: string (includes class and method from its been logged)

* OutputDestination - ENUMS for better extensibiility
	- Console, File, Database: ENUM (for better extensibility)

* ConfiguraitonHolder
	- rootLogLevel: LogLevel
	- destinationApplicable: OutputDestination
